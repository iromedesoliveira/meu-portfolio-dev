# Dashboard de Alocação de Investimentos

Este projeto é uma ferramenta simples de automação para calcular a distribuição de capital baseada na minha estratégia de reinvestimento de lucros de *day trade*.

## Funcionalidades
- **Cálculo Automático**: Recebe o valor total disponível e divide automaticamente conforme a estratégia estabelecida.
- **Distribuição definida**:
    - 30% para Reservas de Emergência/Poupança.
    - 30% para Ações pagadoras de dividendos.
    - 40% para reinvestimento em Day Trade.

## Como utilizar
1. Abra o arquivo `script.js`.
2. Altere o valor passado na função `calcularAlocacao(valor)` para o montante que você deseja distribuir.
3. Execute o script no console do navegador ou via Node.js para visualizar o resultado.

## Objetivo
Manter a disciplina financeira através de cálculos rápidos, garantindo que a alocação de recursos siga rigorosamente o plano de investimentos definido.
